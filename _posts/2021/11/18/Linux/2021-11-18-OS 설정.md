# OS 설정

1. ###### hostname

   - hostname 확인

     ```
     # hostname
     ```

   - hostname 영구 변경

     ```
     hostnamectl set-hostname myhost
     ```





2. ###### Mount

   1) 마운트할 폴더 생성

   2) mkfs.xfs	/dev/nmvn~						# Mount 등록

   3) ll	/dev/disk/by-uuid/						#UUID 확인

   4) vim	/etc/fstab

   ```
   UUID = nmvn의 uuid		/usr/sap(파일경로)		xfs			defaults		0		0
   ```

   5) mount -a





3. vim	/etc/ssh/sshd_config

   /pass 검색으로 숫자 변경
