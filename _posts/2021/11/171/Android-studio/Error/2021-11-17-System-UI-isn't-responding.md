<h1>System UI isn't responding</h1>

> Cause : Allocated emulator device memory etc...



+ Stop & off the emulator device.
+ Delete file(C:\Users(User).android\avd\(Emulator device)avd\userdata-qemu.img).
+ Restart emulator.
