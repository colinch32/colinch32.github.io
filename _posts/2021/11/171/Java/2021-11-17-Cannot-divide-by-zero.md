# Cannot divide by zero

`````kotlin
e.g.
240/(120/160);

//java.lang.ArithmeticException: divide by zero
`````



> :warning: Caution to primitive data type calculation.
>
> Dividing int by int will make int type result.
> -> Cast type float or double.
